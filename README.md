# Careera

JobTrack is a full-stack job application tracking and resume analysis
platform designed for university students and recent graduates.

## Project Objectives

The application will allow users to:

- Create an account and log in
- Add and manage job applications
- Track application stages
- Upload PDF resumes
- Compare resumes with job descriptions
- View job application statistics
- Deploy the final application using AWS
