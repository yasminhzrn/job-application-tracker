# Careera Future Features

**Version:** 1.0

---

# Purpose

This document records ideas that are intentionally excluded from Version 1.

These features may be implemented in future releases after the MVP is completed.

---

# Phase 2 Features

## Resume Version Management

Allow users to maintain multiple resumes for different job types.

Examples:

- Data Analyst Resume
- Business Analyst Resume
- Consulting Resume

---

## Resume Comparison

Compare two resume versions and identify improvements.

---

## Status History

Store every application status update with timestamps.

---

## Interview Management

Record:

- Interview Date
- Interview Type
- Interview Notes
- Interview Outcome

---

## Application Archive

Archive completed applications instead of deleting them.

---

## CSV Export

Export application history for analysis.

---

## Profile Editing

Allow users to update:

- Name
- Email
- Password

---

# Features Excluded from Version 1

The following will not be implemented in the MVP:

- Mobile App
- AI Features
- Recruiter Portal
- LinkedIn Integration
- Google Login
- Microsoft Login
- Calendar Integration
- Email Notifications
- Cover Letter Generator
- Interview Simulator

These features remain candidates for future releases after the MVP is complete.
